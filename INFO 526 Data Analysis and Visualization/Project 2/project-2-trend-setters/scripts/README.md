# Our codes:

This file contains the codes used for analyzing the spatio-temporal effects of climate variables (temperature and precipitation) on corn production and prices in the United States, including inflation-adjusted financial indicators.

# Codebook for our datasets:

## Adjusting_For_Inflation.R

Preprocesses CPI data and generates adjusted prices.

## Seasonal_Price_Dynamics.R: 

Creates multi-axis seasonal plots for precipitation, temperature, and price.

## Stock_vs_Temp.R: 

Produces dual-axis smoothed line plot comparing long-term temperature and stock trends.
