This project is about the study of climate variability and the economics of corn production. 

The work done in this project is uploaded in this repository.

Steps to access different parts of the repo (description of different folders):

README.md: This file contains all the information about different folders in this repository.
proposal.qmd: This file contains out proposal for this project.
about.qmd: This file contains the details about the team members.

Data: This folder holds the datasets used.
Reports: This folder has the .pdf versions of the project reports (proposal and write up).
Plots: This folder contains all the plots obtained for this project.
Code: This folders contains all the codes files (.rmd) for the project.
Presentation: This folder has the .pptx file for the project.
