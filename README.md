# HLT-Capstone-Projects

This repository contains a curated collection of capstone and course projects completed during my **M.S. in Human Language Technology** at the University of Arizona.  

The work spans **Natural Language Processing (NLP), Data Mining, Data Visualization, and Computational Linguistics**, with a focus on building reproducible, data-driven workflows and interpretable results.

---


---

## 📊 Projects

### 📌 INFO 523 — Data Mining and Discovery  
**Focus:** Data mining, exploratory data analysis, and predictive insights  

- Analysis of **spatio-temporal climate effects** (temperature & precipitation)  
- Impact on **corn production and pricing**  
- Data preprocessing, feature engineering, and visualization  
- R-based workflows with reproducible outputs  

---

### 📈 INFO 526 — Data Analysis and Visualization  
**Focus:** Data storytelling and visualization  

- Creation of **interactive and static visualizations**  
- Use of tools such as **R, ggplot2, and Looker Studio**  
- Emphasis on **clear communication of insights**  
- Dashboard design and reporting  

---

### 🧠 LING 581 — Advanced Computational Linguistics  
**Focus:** NLP and computational modeling of language  

- Implementation of **text processing pipelines**  
- Work with **syntactic parsing, feature extraction, and modeling**  
- Exploration of linguistic structure using computational methods
